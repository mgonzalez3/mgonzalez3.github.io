r de Pearson

En esta entrada voy presentar algunas anotaciones del folleto/panfleto de Chen & Popovich (2002). Es un excelente recurso y debería ser una lectura requerida para futuros investigadores en psicología y ciencias sociales. Las anotaciones no están detalladas son más bien una introducción al contenido de este panfleto.

  • La necesidad de utilizar modelos causales en psicología. La estadística utilizada en particular no es la que determina si se pueden realizar inferencias causales, para esto necesitamos un diseño de investigación solido con un buen fundamento lógico.
  • Las correlaciones solo proveen pistas o evidencia a favor o en contra de nuestros modelos. La presencia de una asociación/correlación no es suficiente para concluir que hay una relación causal, pero su ausencia es una pista importante de que no hay causalidad (es necesaria).
  • El motivo para realizar la correlación es importante, ya que de esto dependen los supuestos. Deseamos utilizarlo como una información descriptiva de la muestra, deseamos estimar la correlación poblacional, o probar si la correlación es distinta de 0.
  • La r de Pearson no es una proporción para esto tenemos \(r^2\).
  • La r de Pearson se puede tratar como un tamaño del efecto/relación entre dos variables.
  • A los psicólogos no nos gusta ver formulas o ecuaciones matemáticas, pero revelan mucha información de lo que esta pasando. Por ejemplo todas las formulas equivalentes para la r de Pearson, incluyendo todas sus variantes para casos especiales. Todas estas son equivalentes

\[\begin{equation} \begin{split} r = \frac{Z_{X_i} Z_{Y_i}} {n}\\ = \frac{\sum{\frac{(X_i - \bar{X_i})(X_i - \bar{X_i})}{n}} } {S_XS_Y}\\ = \frac{ \sum{X_i Y_i} \:-\: \frac{\sum{X_i} \sum{Y_i} } {n} } {\sqrt{ (\sum{X_i^2} \:-\: \frac{(\sum{X_i})^2}{n} )(\sum{Y_i^2} \:-\: \frac{(\sum{Y_i})^2}{n} )}} \end{split} \end{equation}\]

Pensar en la correlación de Pearson como el promedio de las puntuaciones Z de las variables observadas es fácil de recordar.

Prueba de Hipótesis para r

  • Los supuestos necesarios son los siguientes:
    • La distribución bi-variada en la población es normal. Para esto sumamos ambas variables y observamos un histograma, gráfico de QQ , o PP.
    • La relación entre las dos variables es lineal. Para esto observamos un gráfico de puntos (scatterplot).
  • La r de Pearson (el valor observado) es un estimador sesgado de la correlación en la población. Para que no tenga un sesgo debemos realizar una correción. El estimador no sesgado es:

\[r_{adj} = \sqrt{ \frac{(n - 1)r^2-1}{n-2} }\] - El error estándar de la r de pearson es:

\[\sigma_r= \sqrt{\frac{(1-\rho^2)^2}{n}}\]

  • La r de Pearson No se aproxima a una distribución normal cuando el valor absoluto de la correlación en la población es mayor de \(\rho = .30\). No importa el tamaño de la muestra.

Prueba T con las r

  • Solo para situaciones donde la hipótesis nula es \(\rho = 0\).
    • La razón de esta limitación es que la r cuando difiere de 0 en la población no tiene una distribución normal.
  • La distribución de t cuando \(\rho = 0\) es:

\[\frac{r\sqrt{n-2}}{\sqrt{1-r^2}}\]

Siempre debemos controlar por el error de familia (Family Wise Error). Mientras más comparaciones realizamos para determinar sí las correlaciones son siginificativas mayor es la probabilidad de cometer un error tipo I.

Transformación de Fisher

  • Para situaciones donde la hipótesis nula es \(\rho >= 0\).
  • Dado que la r de Pearson no aproxima una distribución en las condiciones indicadas anteriormente (\(\rho > .30\)), podemos hacer inferencias con la transformación de Fisher. Sustituimos \(\rho\) por nuestro estimador que es la r de pearson observada

\[z_r =0.5 * log_e(\frac{1+\rho}{1-\rho})\]

El error estándar de la transformación de Fisher es:

\[\sigma_{z_r}=\sqrt{\frac{1}{n-3}}\]

  • Podemos realizar la siguiente prueba estadística y comparar con el valor crítico de z:

\[z = \frac{z_r - z_o}{\sigma_r}\] - También podemos probar si la correlación es distinta de 0 a un nivel \(\alpha = .05\) creando un intervalo de confianza \(z_r \:\pm\: z_{\alpha} * \sigma_{z_r}\). Para hipótesis de ambas direcciones el valor para un \(\alpha = .05\) es de 1.96.

  • Sí las medidas entre cada par de variables son independientes esta prueba es robusta a violaciones de normalidad con \(n \geq 30\).

  • Para reportar los valores \(z_r\) debemos convertirlos a la r de pearson

\[r = \frac{e^{2z_r}-1}{e^{2z_r}+1}\]

Para comparar dos valores de r que provienen de la misma muestra

\[z = \frac{z_{r_x}- z_{r_y}}{\sigma_{r_x} + \sigma_{r_y}}\]

Para comparar más de dos valores de r

  • Para comparar si multiples valores de r difieren entre sí $_1 = _2 = _3 =…= _k $ usamos la distribución de Ji-cuadrado \(X^2\). En este caso la hipótesis alterna es que por lo menos una correlación es distinta.

  • La ecuación es la siguiente para \(k > 1\) y con grados de libertad \(k-1\)

\[X^2 = (\sum(n_k - 3)z_{rk}^2)-\frac{[\sum(n_k - 3)z_{rk}]^2}{\sum(n_k - 3)}\]

Referencias

Chen, P. Y., & Popovich, P. M. (2002). Correlation: Parametric and nonparametric measures (Vols. 7-139, p. 95) [Book]. Sage.
